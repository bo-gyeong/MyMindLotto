### 📢 기 능
---
1. **원하는 숫자로 조합**
    - 선택한 번호만 이용하여 조합
    - 선택한 번호와 다른 번호를 함께 조합
    - 제외번호 설정
    - 내가 만든 조합 상태 저장, 수정 및 삭제
2. **회차별 당첨번호**
    - 역대 당첨번호 조회
    - 회차번호를 검색하여 당첨번호 확인
3. **큐알코드 스캔**
    - 큐알코드 스캔을 통한 당첨확인
4. **번호 저장 및 갱신**
    - 새로고침하여 번호 갱신
    - 번호 저장 및 삭제
